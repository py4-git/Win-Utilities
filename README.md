## System Tweaking & Optimization

**[WinToys](https://apps.microsoft.com/detail/9P8LTPGCBZXD)** - Modern all-in-one tweaker with a clean UI for optimizing Windows settings, managing services, and cleaning junk files.

**[Chris Titus Tech Windows Utility](https://github.com/ChrisTitusTech/winutil)** - PowerShell-based utility for debloating, tweaking, and installing applications on Windows systems.

**[ThisIsWin11](https://github.com/builtbybel/ThisIsWin11)** - User-friendly tool for debloating Windows 11, managing privacy settings, and customizing the interface.

**[Optimizer](https://github.com/hellzerg/optimizer)** - Open-source utility for disabling telemetry, removing bloatware, and improving performance with a clean interface.

**[O&O ShutUp10++](https://www.oo-software.com/en/shutup10)** - Focused specifically on privacy settings with granular control over telemetry and data collection.

**[Winaero Tweaker](https://winaero.com/winaero-tweaker/)** - Comprehensive customization tool covering appearance tweaks to functional modifications.

**[BleachBit](https://www.bleachbit.org/)** - Free and open-source system cleaner designed to free up disk space and maintain privacy.

**[UnChecky](https://unchecky.com/)** - Automatically unchecks unwanted offers during software installations.
